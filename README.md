# 主题编辑器 (Theme Editor)

一款轻量级、可换主题的文本编辑器，基于 **Tauri** + **Vue 3** 构建。

## 功能特性

- 🎨 **16 款精美主题** - 暗色、亮色、节日、自然、赛博朋克等多种风格
- 💾 **设置持久化** - 自动记住主题选择和字体大小
- 🔢 **行号显示** - 实时显示行号，当前行高亮
- 📐 **字体调节** - 滑块控制，支持快捷键（Ctrl +/-）
- 📁 **文件操作** - 打开和保存文本文件
- ⚡ **极速轻量** - 仅约 10MB，原生性能

## 技术栈

- **前端**：Vue 3 + TypeScript
- **后端**：Rust (Tauri)
- **样式**：CSS 变量动态主题
